About

Student Management System

This project is all about building a management system of students for school
In this project a school/college can manage their student details, Here we can add student details and then we can search any student details by just entering its roll number and many more features are available in this project,like deleting and updating student details

STEPS

STEP 1

First we created student.java in which we declare variables and then define method for each input

STEP 2

we created a databse and then we connect our program with the database using drivermanager.get connection method for storing inputs in our database

STEP 3

we create function for each operation the we performed in our project like saving,deleting and updating records

STEP 4

we created webcontent of your project where we use java servlet and created homepage of our project and .jsp file for each operation we perforemed and hosted on local server using apache tomcat

FINISH

we converted our dynamic project to maven project and write pom.xml and hosted on internet using heroku. Fronded of this project we done using html,css and javascript
